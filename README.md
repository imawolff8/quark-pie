夸克派 - 是一款汇聚上千万影视、音乐、动漫、游戏、书籍、教育等各类热门网盘资源的 APP，用户能够很方便的通过 AI 对话方式检索您想要的资源。

## 主要特点

1. 资源多

    资源数量上千万，还在不断增加中

2. 资源新

   每天实时更新，最新资源应有尽有

3. 易检索

   检索简单，容易使用

4. 易分享

   一键分享热门资源给朋友
